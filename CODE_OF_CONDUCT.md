# Code of Conduct — OctoDos

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Standards

### Expected Behavior
- Use this tool **only** for authorized security testing
- Be respectful and constructive in all interactions
- Report vulnerabilities responsibly
- Help others learn about cybersecurity defense
- Follow all applicable laws and regulations

### Unacceptable Behavior
- Using OctoDos for malicious attacks
- Sharing exploit results without authorization
- Harassment, discrimination, or threats
- Publishing others' private information
- Any conduct that constitutes a criminal offense

## Enforcement

Violations may result in:
1. Warning and correction
2. Temporary ban from the project
3. Permanent ban from the project
4. Reporting to appropriate authorities (for illegal activity)

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any public representation of the project.
